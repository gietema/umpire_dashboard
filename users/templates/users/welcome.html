Performance monitoring of your machine learning product.

Start tracking model performance within minutes.

pip install umpire-ai

Example

Give Umpire your model and your api key, and it will hook into TensorFlow's or Pytorch's model.predict().
import umpire-ai
umpire = Umpire(api_key={my_api_key})
model = umpire.track(model, metrics=['confidence'])

Built with privacy in mind

Umpire only stores the final metrics you want to track, so it will never be able to know more than you want about your model.
