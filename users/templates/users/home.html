Here's how to start using umpire.

Install umpire:
pip install umpire-ai
Then, try to use umpire for the first time with your api key. Your API key:
{{ user.api_key }}
Initiate umpire:
import umpire-ai
umpire = Umpire(api_key='{{user.api_key}}')
model = umpire.track(model, model_name='my_model_name', metrics=['confidence'])
Your model should be the object you will use to do a prediction. It doesn't matter if this is a PyTorch or TensorFlow model. Umpire will figure out which one you are using.
Your model_name is just a string to describe the model you are monitoring.
Now, use your model to predict something and refresh this page to see the confidence.
model.predict(img)
